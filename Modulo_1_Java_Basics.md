# Java Basics

## Objetivos de Aprendizaje (OAs)
1. Sintaxis básica de Java.
2. Manipulación de tipos de datos primitivos.
3. Uso de condicionales.
4. Uso de loops.
5. Trabajo con arrays
6. Trabajo con strings.
